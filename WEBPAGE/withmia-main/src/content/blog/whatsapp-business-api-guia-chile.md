WhatsApp es la aplicación de mensajería más utilizada en Chile, con una penetración del **92% entre usuarios de smartphone**. Para las empresas, esto representa una oportunidad única de comunicarse con sus clientes en el canal que ya usan diariamente.

## WhatsApp Business App vs WhatsApp Business API

Es fundamental entender la diferencia:

### WhatsApp Business App (Gratuita)
- Diseñada para **pequeños negocios**
- Un solo dispositivo / máximo 4 dispositivos
- Respuestas rápidas limitadas
- Sin integraciones con CRM o IA
- Catálogo básico de productos

### WhatsApp Business API
- Para **empresas medianas y grandes**
- **Usuarios ilimitados** simultáneamente
- Integración con CRM, ERP y plataformas de IA
- Automatizaciones avanzadas y chatbots
- Plantillas de mensajes aprobadas
- Analíticas detalladas
- Certificado con **verificación de empresa** (check verde)

## Requisitos para Obtener la API

1. **Cuenta de Facebook Business** verificada
2. **Número de teléfono dedicado** (no puede estar en WhatsApp personal)
3. **Proveedor de soluciones (BSP)** autorizado por Meta
4. **Política de privacidad** publicada en tu sitio web
5. **Cumplimiento** con las políticas comerciales de WhatsApp

## Modelo de Precios

WhatsApp Business API utiliza un modelo de **precios por conversación**:

| Tipo de Conversación | Descripción | Costo Aprox. (Chile) |
|----------------------|-------------|---------------------|
| Marketing | Promociones, ofertas | ~$0.05 USD |
| Utilidad | Confirmaciones, actualizaciones | ~$0.03 USD |
| Autenticación | Códigos OTP, verificación | ~$0.02 USD |
| Servicio | Iniciadas por el cliente | Gratis* |

*Las conversaciones de servicio (iniciadas por el cliente) tienen una cuota gratuita mensual. Meta actualiza precios periódicamente.
