

## Clonar el repositorio

```bash
# Clona el repositorio (reemplaza la URL por la del proyecto)
git clone https://github.com/usuario/nombre-del-repo.git

# Entra en la carpeta del proyecto
cd nombre-del-repo
```

---

## Instalación de dependencias

Usando npm:

```bash
npm install
```

O usando yarn:

```bash
yarn install
```

```bash
# Levantar en modo desarrollo (hot reload)
npm run dev

# Compilar/crear build para producción
npm run build
